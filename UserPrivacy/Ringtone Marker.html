Data collected and used by Ringtone Marker

  Ringtone Marker does not ask for users to register personal account or login with an account. Just download and one tap to connect. Users will not be worried about personal information security.
  
  Except for necessary data for routine purpose that we collect and store to operate the service, Ringtone Marker does not record any personal information or sensitive data, like contacts, address, cellphone number etc.
  
  Like all other applications will do so, the general data information that Ringtone Marker collected from users includes:
  * Detect user network state for network operations and support
  * Allow application to connect network, so that it can send data to our server.
  
  Ringtone Marker keeps no logs which enable interference with your IP address, the moment or content of your data traffic. We make express reference to the fact that we do not record in logs communication contents or data regarding the accessed websites or the IP addresses. 
  
  What’s more, all user data are encrypted. We offer private and secure data transmission when using Ringtone Marker. Therefore, the user’s personal data is not monitored, recorded, logged or stored.
